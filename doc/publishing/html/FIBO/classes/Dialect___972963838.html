  • definition "a language variant specific to a geographical region or a group of language users" (string)
  • note "The dialect of a language is usually represented by the same language code as that used for the language. If the language is assigned to a collective language code, the dialect is assigned to the same collective language code. If the language has an individual language code, the dialect is also included in that code rather than the code for the group to which both belong. In a few instances, however, both the language and a dialect of that language have their own individual language codes." (string)
