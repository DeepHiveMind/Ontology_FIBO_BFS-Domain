Annotations (3)

  • explanatoryNote "This property should be read as always being a property of some kind of 'relative thing', that is a thing defined in some context. Generally this will be a 'party in role'. This property is not intended to be used to relate some independent thing to that which it manages, instead it must only be a property of something in the role of being that which manages some thing." (string)
  • label "manages" ()
  • definition "relates an autonomous agent to something that it directs in some way" (string)
