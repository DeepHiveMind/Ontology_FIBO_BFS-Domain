  • definition "a language identifier for a human language as defined by ISO 639, that is, an alpha-3 (three-letter) combination of characters used to represent a language or languages" (string)
  • note "The language codes consist of three Latin-alphabet characters in lowercase. No diacritical marks or modified characters are used. Implementors should be aware that these codes are not intended to be an abbreviation for the language, but to serve as a device to identify a given language or group of languages. Thus, in this ontology, alpha-3 code is modeled as a subclass of designation rather than abbreviation, which might seem more natural. The language codes are derived from the language name. Two code sets are provided in ISO 639-2, one for bibliographic applications (ISO 639-2&47;B), and one for terminology applications (ISO 639-2&47;T). Some of these are considered collective language codes, referring to a family of languages, and there are a few special purpose codes for use in various applications. ISO 639-3 provides a single, three character code for each individual language or macrolanguage. Additional codes for language families and groups may be specified in anticipated forthcoming parts of the standard." (string)
