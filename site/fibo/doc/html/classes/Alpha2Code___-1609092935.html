Class: Alpha2Code

http://www.omg.org/spec/LCC/Languages/LanguageRepresentation/Alpha2Code

Annotations (5)

  • directSource "Section 4.1, ISO 639-1" (string)
  • label "alpha-2 code" ()
  • definition "a language identifier for a human language as defined by ISO 639, that is, an alpha-2 (two-letter) combination of characters used to represent a language or languages" (string)
  • note "In the language code ISO 639-1, each language identifier is composed of two letters. The language identifiers consist of the following 26 letters of the Latin alphabet in lower case - a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z. No diacritical marks or modified characters are used." (string)
  • scopeNote "4.1 (639-1) Implementors should be aware that these identifiers are not intended to be an abbreviation for the language, but to serve as a device to identify a given language. The language identifiers are derived from the language name. Each identifier is based on the indigenous name of the language or the preference of the communities using the language." (string)
