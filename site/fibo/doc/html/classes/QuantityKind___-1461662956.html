  • explanatoryNote "Every instance of ‘quantity kind’ is also a specialization of ‘quantity’. So the concept ‘duration’ is an instance of ‘quantity kind’ and it is a specialization of ‘quantity’, i.e., it is a classifier of actual quantities. But a given duration (i.e., the duration of something) is an instance of ‘duration’ and thus a ‘particular quantity,’ not an instance of ‘quantity kind’. For example, a ‘year’ is not an instance of quantity kind; it is an instance of quantity, but not a category of quantity." (string)
