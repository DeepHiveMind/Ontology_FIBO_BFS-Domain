Annotations (3)

  • label "alpha-2 code" ()
  • definition "the digraph representing the country name" (string)
  • note "This part of IS0 3166 establishes an alphabetic 2-character (alpha-2) code which is generally recommended to represent country names and which is the basis for the codes laid down in IS0 3166-2 and IS0 3166-3 and for other international standards and recommendations. The alpha-2 code uses combinations, in upper case, of two letters of the 26-character Roman alphabet (ignoring diacritic signs) from the range AA to ZZ." (string)
