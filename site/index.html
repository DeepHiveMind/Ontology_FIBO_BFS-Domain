EDM Council - Specifications

FIBO

The EDM Council is publishing FIBO quarterly on https://spec.edmcouncil.org/fibo. This includes 400 Plus files that are all of the 30 Financial Industry Domain/subdomain ontologies that have been being developed for many years.

Much of these 30 Domains will serve as raw content to be further expanded and refined by Subject Matter Experts using the FIBO Build-Test-Deploy-Maintain Methodology in a continuous development process goverened by automation including the Council developed open source RDF ToolKit, GitHub, Jenkins, SourceTree and a number of Wiki's for collaboration.

Of immediate importance, within these 30 Domains and Subdomains are 8 Domains that have been vetted by SMEs, and have passed standard industry Hygiene tests for Web Ontology Language (OWL) content.

In addition to content, spec.edmcouncil.org/fibo will contain a FIBO Navigator. This will enable both casual and expert users to pick and choose FIBO content and how they want to visualize and use that content. spec.edmcouncil.org/fibo and the FIBO Navigator are works in process.

The EDM Council welcomes and encourages participation from users and developers. Look around:

GitHub The EDM Council GitHub Repositories All FIBO Content and Tools are here. Some of it is open and some is private.
JIRA The FIBO Issue Manager There is JIRA Project for every FIBO Domain and for every FIBO Team. It is necessary to request to be added to JIRA.
Wiki's There is a Wiki for every FIBO Domain and Team. These wiki's correspond to the JIRA projects When a person is added to JIRA, he or she is automatically connected to the corresponding Wiki.

For more information on any of the above, please contact. dwisnosky@edmcouncil.org.
